
# Initialisations 

```
npm init
```

- Core npm React
```
npm i react react-dom react-scripts react-router-dom
```

- Core Redux toolkit
```
npm i @reduxjs/toolkit react-redux
```
- i18n
```
npm install react-i18next i18next
```

- Autres
```
npm i --save-dev sass typescript gh-pages @types/react-dom
```
